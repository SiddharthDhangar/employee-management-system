# Employee Management System

A web-based Employee Management System to manage employee data, task assignment, and role-based access with a clean and user-friendly interface.

## Features
- Add, update, and delete employee records
- Task assignment and tracking
- Role-based access
- Responsive UI

## Tech Stack
- React.js
- JavaScript
- HTML, CSS
- Local Storage / API

## Getting Started
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
